# Big Data Artificial Intelligence Club Website

This repository hosts the official website for the Big Data Artificial Intelligence Club at the University of Baghdad.  
It is a student-led initiative to explore Big Data and Artificial Intelligence through workshops, projects, and community events.


# Features
- Responsive design (works on desktop, tablet, and mobile).
- Navigation bar linking all pages.
- Extended Home page with About, Activities, Events, and Testimonials.
- Contact form with Name, Email, Message fields and a Submit button.
- Clean and modern styling with CSS.
